# Vonk-E-Racing

Racing app mockup design built with React, TypeScript, and Vite.

The original project is available at https://www.figma.com/design/NF5b9TjU0ZCRStSqnu9m6l/Racing-app-mockup-design.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
